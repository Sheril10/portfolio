My Projects

Project 1: Rickshaw Management System (SSSS)

Introduction: This project was done with the Society for Sustainable Social Solutions (SSSS) to empower marginalized communities in Pakistan by distributing rickshaws. It focuses on sustainable livelihoods, education, and economic stability.

Project Domain: Rickshaw Management System

Objectives:

  • User-friendly interface
  • Record keeping for rickshaw distribution
  • Admin/Client dashboards, financial tracking
  • Scalable system for future updates

Risks: Privacy, Financial tracking, Data Security

Conclusion: A key solution to empower communities and improve management transparency in SSSS’s rickshaw distribution mission.

Project 2: E-Commerce Platform on DBMS

This platform allows us to browse, order, and review products, with full backend support for orders, payments, shipments, and product management on Database Management System.

Core Features: Customer profiles, order placement, payment methods, reviews, ratings, staff management, and shipment tracking.

Total Records: 8102 entries across users, products, orders, and reviews.

Project 3: Line Following Robot Car

This project involved making a robot car using AND/OR/NOT gates, DC motors, and sensors. It self-detects and follows a black line track, including turns, built in the Digital Logic Design course.
